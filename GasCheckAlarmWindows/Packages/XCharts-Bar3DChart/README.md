# XCharts-Bar3DChart

XCharts的付费扩展图表 - Bar3DChart 3D柱图。

## 使用教程

1. 先确保项目里有`XCharts`
2. 再将`XCharts-Bar3DChart`放到和`XCharts`同目录下即可使用
